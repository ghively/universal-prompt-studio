Prompt Studio · Design direction

Specimen & margin

The prompt is a specimen — an exact payload bound for a machine, set in monospace on cool paper. Everything human — explanation, coaching, questions — lives in the margins in editorial type. Cobalt is the single accent; green, amber, and red speak only when evidence does.

Palette and type, chosen not defaulted

Cool paper with a blue bias toward the accent. Three type voices: a bookish serif for narrative, a quiet sans for UI, and mono for anything that is data — payloads, costs, checks, model IDs.

Rules

Four principles the UI never breaks

Mono means measured

Monospace is reserved for payloads and evidence — costs, checks, model IDs, versions. If it's mono, you can trust it was counted, not written.

Semantic color is earned

Green, amber, and red appear only as verdicts from real runs. Decoration never borrows them; the accent never judges.

Margins teach

Explanation sits beside the artifact, not in a modal or a tooltip graveyard. The annotated specimen is the curriculum.

Calm by default

Nothing pulses, spins, or slides unless state actually changed. Motion is a reporting tool, honored only when the viewer allows it.
